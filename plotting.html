fastf1.plotting - Plotting module¶

Helpers for charming graphs.

fastf1.plotting.team_color(identifier)[source]¶

Get a teams color.

Parameters

identifier (str) – Abbreviation or full name of the team. For example “RBR” or “Red Bull” would both return the same result.

Returns

hex color code

Return type

str

fastf1.plotting.timedelta_converter(x)[source]¶

Special data type converter for laptime axis.

Masks NaT and NaN values in data arrays and converts nanoseconds based times to seconds if necessary. Other data types but np.ndarray are left unchanged.

fastf1.plotting.laptime_axis(ax, axis='yaxis')[source]¶

Convert axis to change time formatting from seconds to “mm:ss.ms”

Parameters

  • ax – matplotlib axis

  • axis (='yaxis', optional) – can be ‘xaxis’ or ‘yaxis’
