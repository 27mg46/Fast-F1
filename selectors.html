fastf1.selectors - Selectors module¶

This module provides a set of functions to be used in combination with fastf1.core.Lap.sel().

fastf1.selectors.driver(name)[source]¶

Select driver given his three letters identifier

fastf1.selectors.drivers(names)[source]¶

Select drivers given a list of their three letters identifiers

fastf1.selectors.team(name)[source]¶

Select team given its name

fastf1.selectors.teams(names)[source]¶

Select teams given a list of names

fastf1.selectors.fastest(x)[source]¶

Select fastest lap time

fastf1.selectors.quicklaps(x)[source]¶

Select laps with lap time below QUICKLAP_THRESHOLD (default 107%) of the fastest lap from the given laps set
